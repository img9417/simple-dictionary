# Simple English Dictionary

## スタートガイド

### 1. リポジトリのクローン

```bash
git clone https://github.com/rolemadelen/simple-dictionary.git
```

プロジェクトディレクトリへ移動。

```bash
cd simple-dictionary/react/
```

### 2. Dependecy をインストール

```bash
pnpm install
```

または

```bash
npm install
```

### 開発サーバを立ち上げる

```bash
pnpm run dev
```

または

```bash
npm run dev
```

`localhost:5173`で確認できます。
